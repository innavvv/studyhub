# StudyHub - навчальна платформа

Це навчальний проект, створений у межах лабораторних робіт з веб-розробки.  

# Використані технології:

# СSS

- Flexbox
- CSS Grid
- Адаптивний дизайн (media queries)
- Компонентна структура стилів:
  - `base.css` - базові стилі та типографіка
  - `layout.css` - розмітка сторінок та структурні блоки
  - `components.css` - кнопки, картки, форми
  - `responsive.css` - адаптивність сайту
- Валідація форм (CSS + HTML)
- Бургер-меню

# JavaScript
- Розкриття бургер-меню
- Перевірка ширини хедера
- Клієнтська валідація контактної форми

# HTML
- Семантична структура
- Використання grid та flex у макеті
- Підключення зовнішніх стилів і скриптів

# Адаптивність

Сайт коректно відображається на:
- мобільних телефонах (360px+)
- планшетах (768px+)
- ноутбуках та ПК (1024px+)
